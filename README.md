# Blog Post

Projeto de página de artigo desenvolvido para praticar conceitos fundamentais de desenvolvimento Front-end com HTML e CSS.

## Recurso e Funcionalidades

* Estrutura HTML semântica
* Banner principal do artigo
* Cartão com informações do autor
* Layout responsivo para telas menores
* Estilização modular e organizada

## Tecnologias Utilizadas

* HTML5
* CSS3

## Aprendizados

Durante a construção desta página, os seguintes conceitos foram colocados em prática:

* Organização e estrutura de arquivos do projeto
* Uso de classes utilitárias no CSS
* Implementação de design responsivo
* Ajuste e manipulação de imagens utilizando a propriedade `object-fit`
* Separação de conteúdo usando tags semânticas (`header`, `main`, `article` e `footer`)

## Como Executar o Projeto

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/seu-repositorio.git

```


2. Acesse o diretório do projeto:
```bash
cd seu-repositorio

```


3. Abra o arquivo `index.html` diretamente no seu navegador de preferência.
